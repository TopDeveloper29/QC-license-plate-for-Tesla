# QC-license-plate-for-Tesla
Repository containing a Québec (QC) license plate image sized specifically for Tesla’s custom wrap / license plate feature. The plate number is removed while keeping the official text, logos, and borders.

## Documentation
Prepare your USB device by creating a folder named 'LicensePlate'. 
Inside the folder, add up to 10 custom '.png' background images.

- Each image should be under 0.5 MB in size.
- File name should be under 32 characters long with letters and numbers only.
- Each image should have a resolution of 420 x 200 pixels.
- Insert the prepared USB device into the USB port in the vehicle and navigate to Background -> Image.

If the files on the USB flash drive meet the requirements, then the custom images will be available to select from the menu.
